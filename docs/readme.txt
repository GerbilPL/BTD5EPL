Translation files for BTD 3.37 and BTD5 Expansion 12. Version pre-release 1

Credits: GerbilPL - main translator, pelletsstarPL - for feedback and help.

How to install:
- Make sure you have installed BTD5 Expansion 12 mod by 5h3i1ah.
- Make sure that game version matches info at the top.
* Optional: Make backup of English.xml file (Located in Assets\Loc).
- Move Entire Assets folder from zip file to main bloons directory, and say yes to replacing/overwriting files.
- Or move each file from zip to its corresponding location in the BTD install.
- Done.

Jak zainstalować:
- Upewnij się że masz zainstalowaną modyfikację BTD5 Expansion 12 stworząną przez 5h3i1ah.
- Upewnij się że wersja gry odpowiada informacji u góry.
* Opcjonalne: Zrób kopię zapasową pliku English.xml (Jest w Assets\Loc).
- Przenieś cały folder Assets z zipa do głównego katalogu bloonsów, i zastąp/nadpisz pliki.
- Lub przenieś każdy plik z zipa do jego odpowiadającej lokalizacji w instalacji BTD.
- Gotowe.
