POLSKI:

Pliki spolszczenia dla BTD5 4.4 i BTD5 Expansion 12. Wersja: Genesis

Zasługi: GerbilPL - główny tłumacz, pelletsstarPL - pomoc w tłumaczeniu, cosmo - pomoc w tłumaczeniu

Jak zainstalować:
- Upewnij się że wersja gry odpowiada informacji u góry.
- Upewnij się że masz zainstalowaną modyfikację BTD5 Expansion 12 stworząną przez Shaiko.
* Opcjonalne: Zrób kopię zapasową pliku English.xml (Znajdującego się w Assets\Loc).
- a) Przenieś cały folder Assets z pliku zip do głównego katalogu gry, i zastąp/nadpisz pliki.
- b) Lub przenieś każdy plik z zipa do jego odpowiadającej lokalizacji w instalacji BTD5.
- Gotowe.


ENGLISH:

Polish translation files for BTD5 4.4 and BTD5 Expansion 12. Version: Genesis

Credits: GerbilPL - main translator, pelletsstarPL - translation help, cosmo - translation help

How to install:
- Make sure that game version matches info at the top.
- Make sure you have installed BTD5 Expansion 12 mod by Shaiko.
* Optional: Make backup of English.xml file (Located in Assets\Loc).
- a) Move Entire Assets folder from zip file to main game directory, and say yes to replacing/overwriting files.
- b) Or move each file from zip to its corresponding location in the BTD5 install.
- Done.
